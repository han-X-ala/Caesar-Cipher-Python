# Caesar Cipher Project

A simple Python program that encrypts and decrypts text using the Caesar Cipher technique.

## Features
- Encrypt text using shift
- Decrypt text using shift
- Input validation for shift values
- Handles uppercase and lowercase letters

## How to run
1. Run the Python file: python caesar_cipher.py
2. Enter text, shift value, and mode (e/d)

## Example
Input:
hello, shift = 3, mode = e

Output:
khoor
